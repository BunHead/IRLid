Community

IRLid is built in the open. Whether you're using it, running check-ins at a venue, poking holes in the protocol, or just curious — here's where to talk to us and to each other.

💬 Discussion forum

Questions, ideas, deployment patterns, and protocol design happen in the open on GitHub Discussions — our community forum. If you have a GitHub login, you're in; no separate account to set up.

IRLid is open source. Read the code, star the repo, or file a bug or feature request as a GitHub Issue.
