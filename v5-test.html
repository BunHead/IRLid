Step 3 · Sign a test payload

Builds a fake-but-realistic IRLid payload, hashes it, and signs the hash via your enrolled v5 credential. Another biometric prompt is expected.

Step 4 · Verify the v5 envelope

Runs the signed envelope from Step 3 through irlidV5VerifyEnvelope() — the same verifier the Worker uses. No biometric prompt.

Step 5 · Full round-trip via the dispatcher

Toggles the v5-enabled flag, runs irlidSignPayload() (the production code path), verifies the result, then restores the flag. Proves the integration works end-to-end.

Removes the v5 credential reference from this device. Optional. The platform-side credential remains in your password manager until you delete it there.
