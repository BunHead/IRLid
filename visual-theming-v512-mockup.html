Polish carry-overs (Captain's 28 May review)

  • Ripple effect — border around frame looks odd. Effect's outer radial gradient painting past the orange-dome rounded bottom corners. Fix: border-radius: inherit; overflow: hidden on the effect layer.
  • Vignette flash — same border issue. Same clip-to-stage fix.
  • EDITING: DENY pill — magenta border framing the live preview during Deny mode editing might distract.
  • Inline-vs-fullscreen offset — store anchor positions as percentage offsets from anchor target's centre, not absolute pixels. Same percentage renders correctly at any stage size.
  • W3C accessibility — anchor name tooltips use aria-describedby for WCAG 2.1 SC 3.3.2, 1.3.1, 3.2.4.
  • Ctrl+Z / Ctrl+Y undo stack — 50-entry ring buffer. Per-effect "Default" button is narrower scope (reset one effect's settings).
